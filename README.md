## 

### 如何运行项目
1. 运行环境  
    <u>Java1.8</u>  
    <u>MySQL 8.0 </u>  
    <u>Apache Maven 3.5.3</u>  
    <u>Spring Boot 2.1.0</u> 
2. 初始化数据库 执行init.sql
3. 修改application-dev.yml配置，设置IDE开发环境为UTF-8
4. 运行相应模块Application
